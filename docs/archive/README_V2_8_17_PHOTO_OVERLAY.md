# GCFind v2.8.17 Photo Overlay Fix

Changes:
- Item detail photo now opens in an in-page overlay/modal instead of a new tab.
- Overlay includes title, full photo preview, close button, outside-click close, and Escape key close.

Testing:
1. Open `details.html?id=<item_id>`.
2. Click the item photo.
3. Confirm full preview opens inside the same dashboard page.
4. Close using X, Escape, or clicking the dim background.
